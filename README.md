# RacingDeck

Elgato Stream Deck plugin to dynamically update actions based on simulator state.

## Actions

Currently supported actions with current state display:

* **Headlights** - hotkey: L
* **Rain Light** - hotkey: Ctrl+L
* **Wipers** - hotkey: Alt+R
